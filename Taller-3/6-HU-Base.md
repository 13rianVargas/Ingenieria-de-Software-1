## Guía: Serie de Fibonacci para estimaciones

En Planning Poker usamos la serie de Fibonacci modificada para estimar el esfuerzo relativo de cada historia (1, 2, 3, 5, 8, 13, 20, 40, 100). Cada número representa el tamaño relativo, donde valores más altos indican mayor complejidad e incertidumbre.
