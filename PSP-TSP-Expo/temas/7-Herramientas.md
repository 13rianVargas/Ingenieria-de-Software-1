# 7. Herramientas

## Personal Software Process (PSP)

#### Time Logging (Registro de Tiempos)
Excel/Google Sheets: fase, hora inicio/fin, acumulado. Registro real vs. estimado.

#### Defect Tracking (Seguimiento de Defectos)
Tipo, fase introducida/detectada, severidad. Base para calcular Yield (efectividad).

#### PROBE Calculator
Estima LOC, esfuerzo y tiempo usando datos históricos. Regresión lineal → rango de confianza.

#### Process Scripts (Guiones)
Checklist estándar: planeación, diseño, revisión, codificación, pruebas. Evita omisiones.

#### Quality Metrics Dashboard
Gráficas de Yield, defectos por fase, productividad. Visualiza tendencias de mejora personal.

---

## Team Software Process (TSP)

#### Project Tracker / Work Dashboard
Jira, Azure DevOps, Trello: tareas, responsable, % completado, bloqueadores. Actualización diaria.

#### Daily Stand-up
Reunión 15 min: ¿qué hice? ¿qué haré? ¿obstáculos? Detecta bloqueadores temprano.

#### Configuration Management System (Git)
Versionado, ramas, etiquetado, política de merge. Trazabilidad + colaboración segura.

#### Issue / Defect Tracker
Jira, Azure: severidad, prioridad, asignado, estado. Ciclo: detección → resolución → verificación.

#### Metrics Aggregator
Consolida PSP → equipo: burndown, velocidad, defects, cobertura. Guía decisiones.

#### Continuous Integration / Deployment (CI/CD)
Jenkins, GitHub Actions: compilación automática, pruebas, análisis, despliegue. Feedback rápido.

#### Test Automation
JUnit, Selenium, Cypress: pruebas unitarias, integración, funcionales en CI/CD.

#### Risk Management + Retrospective
Registro centralizado de riesgos + sesiones de lecciones aprendidas al cierre de ciclos.
